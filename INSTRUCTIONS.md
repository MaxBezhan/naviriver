## Системний адміністратор

### Повноваження
- Повний доступ до всіх функцій системи
- Управління користувачами всіх ролей
- Управління навчальними закладами
- Налаштування системи

### Основні функції

#### 1. Управління користувачами
- **Додавання користувачів**: Адмін-панель → Користувачі → Додати
  - Вкажіть ім'я користувача, ПІБ, контакти
  - Оберіть роль (Слухач, Викладач, Адміністратор школи)
  - Встановіть термін дії (безстроково, кількість днів, до дати)
  - Призначте заклад (обов'язково для Викладача та Адміністратора школи)
  - **Пароль за замовчуванням: `Navirver2024!`**
  - Користувач повинен змінити пароль при першому вході

- **Редагування користувачів**: Натисніть ✏️ поруч з користувачем
  - Зміна анкетних даних
  - Зміна ролі
  - Зміна терміну дії
  - Призначення/зміна закладу

- **Скидання пароля**: Натисніть 🔑 поруч з користувачем
  - Генерується новий пароль
  - Користувач повинен змінити пароль при наступному вході

- **Активація/деактивація**: Натисніть 🚫/✓ поруч з користувачем

#### 2. Управління закладами
- **Додавання закладу**: Адмін-панель → Заклади → Додати
  - Назва, адреса, телефон, email

#### 3. Імпорт/експорт користувачів
- **Експорт**: Кнопки 📤 JSON / 📤 CSV
- **Імпорт**: Завантажте JSON файл з профілями користувачів

#### 4. Управління питаннями
- Доступ до редактора питань
- Імпорт/експорт питань
- Додавання, редагування, видалення питань

## Адміністратор школи

### Повноваження
- Управління користувачами свого закладу
- Перегляд статистики слухачів свого закладу
- Редагування питань
- Активація/деактивація слухачів

### Основні функції

#### 1. Управління слухачами
- Перегляд списку слухачів свого закладу
- Активація аккаунтів слухачів
- Продовження терміну дії аккаунтів
- Скидання паролів

#### 2. Редактор питань
- Додавання нових питань
- Редагування існуючих питань
- Призначення категорій (МС - малі судна, ГЦ - гідроцикли)
- Завантаження зображень до питань
- Вказівка правильної відповіді (1, 2 або 3)

#### 3. Імпорт/експорт питань
- Імпорт з JSON/CSV файлів
- Експорт в JSON/CSV формати

#### 4. Статистика
- Перегляд результатів тестування слухачів
- Аналіз помилок
- Відстеження прогресу

## Викладач

### Повноваження
- Редагування питань
- Перегляд статистики слухачів
- Імпорт/експорт питань

### Основні функції

#### 1. Редактор питань
- **Додавання питання**:
  1. Оберіть розділ
  2. Введіть текст питання
  3. Введіть 3 варіанти відповіді
  4. Вкажіть правильну відповідь (1, 2 або 3)
  5. Оберіть категорії (МС, ГЦ) або залиште порожнім для всіх
  6. За потреби завантажте зображення

- **Редагування питання**:
  1. Знайдіть питання в списку
  2. Натисніть ✏️
  3. Внесіть зміни
  4. Збережіть

#### 2. Категорії питань
- **МС (Малі судна)** - питання для категорії малих суден
- **ГЦ (Гідроцикли)** - питання для категорії гідроциклів
- Питання без категорії доступні для всіх

#### 3. Режими тестування
- **По розділах** - питання з конкретного розділу
- **Випадкові** - випадковий набір питань
- **Іспит** - 10 питань, 20 хвилин
- **За категорією** - фільтр за МС або ГЦ
- **Помилки** - питання з помилками
- **Непройдені** - нові питання
- **Навчання** - з показом правильних відповідей

#### 4. Налаштування тесту
- Таймер (без обмеження / 2 хвилини на питання)
- Перемішування питань
- Перемішування відповідей
- Показ результату (відразу / в кінці)

## Слухач

### Реєстрація та перший вхід

#### 1. Отримання облікових даних
- Отримайте логін та пароль від адміністратора
- Пароль необхідно змінити при першому вході

#### 2. Перший вхід
1. Увійдіть в систему
2. Заповніть профіль:
   - Прізвище
   - Ім'я
   - По батькові
   - Номер телефону
   - Email
3. Погодьтеся на обробку персональних даних
4. Аккаунт активується на 10 днів

### Режими навчання

#### 1. Тестування
- Оберіть режим (випадкові, по розділах, іспит тощо)
- Налаштуйте параметри
- Почніть тестування
- Відповідайте на питання (клік або клавіші 1-3)
- Переміщуйтесь клавішами ← →
- Переходьте до питання кліком на номер в карті

#### 2. Іспит
- 10 питань
- 20 хвилин на весь іспит
- Без повторення питань
- Результат відображається в кінці

#### 3. Навчання
- Показується питання та правильна відповідь
- Можна завантажити всі питання (JSON/CSV)
- Немає оцінювання

#### 4. Режим помилок
- Показуються питання, де були помилки
- Допомагає вивчити складні теми

### Перегляд статистики
- Історія тестів
- Середній бал
- Кількість пройдених питань
- Питання з помилками

### Профіль
- Перегляд та редагування особистих даних
- Завантаження фото профілю
- Зміна пароля

### Термін дії аккаунта
- Аккаунт активується на 10 днів після першого входу
- Після закінчення терміну - зверніться до адміністратора

## Загальні поради

### Навігація клавіатурою
- **1, 2, 3** - вибір відповіді
- **←** - попереднє питання
- **→** - наступне питання
- **Клік на номер** - перехід до питання

### Мобільна версія
- Додаток адаптований для смартфонів та планшетів
- Touch-friendly елементи
- Оптимізований інтерфейс

### Збереження прогресу
- Відповіді зберігаються автоматично
- Можна повернутися до попередніх питань
- Питання позначаються в карті
