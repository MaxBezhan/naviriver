# ⚓ Тренажер для судноводіїв

Веб-додаток для тестування та навчання судноводіїв малих суден.

## 📋 Вимоги

- **ОС**: Windows 10/11 (рекомендується Windows 11)
- **Python**: 3.10+ (рекомендується 3.14.3)
- **Оперативна пам'ять**: 4 GB+
- **Вільне місце**: 500 MB

## 🚀 Швидкий старт

### 1. Встановлення

```powershell
# Відкрийте CMD (не PowerShell!) і виконайте:
cd C:\allin_trainer
python -m venv venv
venv\Scripts\activate.bat
pip install Flask==3.0.3 Flask-SQLAlchemy==3.1.1 Flask-Login==0.6.3 Flask-WTF==1.2.1 Werkzeug==3.0.3 PyJWT==2.8.0 bcrypt==4.1.3 python-dotenv==1.0.1 waitress==3.0.0
python run.py
```

### 2. Відкрийте в браузері
```
http://localhost:5000
```

## 👥 Ролі користувачів

| Роль | Можливості |
|------|-----------|
| **Системний адміністратор** | Повний доступ, управління всіма користувачами |
| **Адміністратор школи** | Управління користувачами закладу, питаннями |
| **Викладач** | Редагування питань, статистика |
| **Слухач** | Тестування, перегляд статистики |

## 🎯 Режими тестування

- **📑 По розділах** - питання з конкретного розділу
- **🎲 Випадкові питання** - випадковий набір
- **📝 Іспит** - 10 питань, 20 хвилин
- **🏷️ За категорією** - МС або ГЦ
- **❌ Помилки** - питання з помилками
- **🆕 Непройдені** - нові питання
- **📖 Навчання** - з показом правильних відповідей

## 📥 Імпорт питань

### Формат JSON
```json
[
  {
    "section": "1",
    "text": "Текст питання",
    "options": ["Варіант 1", "Варіант 2", "Варіант 3"],
    "correct": 0,
    "categories": ["MS"],
    "image": null
  }
]
```

### Формат CSV
```csv
section;question;option1;option2;option3;correct;categories
1;Текст питання;Варіант 1;Варіант 2;Варіант 3;0;MS
```
